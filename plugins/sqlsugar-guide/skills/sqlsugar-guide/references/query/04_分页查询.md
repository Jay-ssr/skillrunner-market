# 分页查询

### 基本分页

```csharp
// API 说明
.ToPageList(pagenumber, pageSize)                        // 不返回 Count
.ToPageList(pagenumber, pageSize, ref totalCount)       // 返回 Count
.ToPageList(pagenumber, pageSize, ref totalCount, ref totalPage)  // 返回 Count + 总页数

// 单表分页
int pageIndex = 1;   // 页码从 1 开始，不是从 0 开始
int pageSize = 20;
int totalCount = 0;
var page = db.Queryable<Student>()
    .ToPageList(pageIndex, pageSize, ref totalCount);

// 多表分页
var list = db.Queryable<Student>()
    .LeftJoin<School>((st, sc) => st.SchoolId == sc.Id)
    .Select((st, sc) => new { Id = st.Id, Name = st.Name, SchoolName = sc.Name })
    .ToPageList(pageIndex, pageSize, ref totalCount);

// SQL
SELECT * FROM
(SELECT [ID],[SchoolId],[Name],[CreateTime],ROW_NUMBER()
OVER(ORDER BY GetDate())AS RowIndex FROM [STudent]) T
WHERE RowIndex BETWEEN 1 AND 20
```

### 异步分页

```csharp
RefAsync<int> total = 0;  // REF 和 OUT 不支持异步，这是最优解
var page = await Db.Queryable<Order>()
    .ToPageListAsync(pageIndex, pageSize, total);
```

### Oracle 高性能分页

```csharp
// 新功能：5.1.2.6-preview03
// Oracle 分了 2 种分页，特殊情况下性能慢可以换下面这种
var page = db.Queryable<Student>().ToOffsetPage(pageIndex, pageSize, ref totalCount);
```

### OFFSET 分页

```csharp
// SqlServer 2012+ 分页
// 把 ToPageList 换成 ToOffsetPage (offset 分页)
var page = db.Queryable<Student>().ToOffsetPage(pageIndex, pageSize, ref totalCount);

// 如果 SqlServer 不想有 RowNumber 也可以用这个
```

### 获取行号

```csharp
// 方式 1：只支持 SqlServer 或 Oracle (row_index)
[SugarColumn(IsIgnore = true)]  // 需要加上
public int RowIndex { get; set; }  // 行号 序号

db.Queryable<Student>().ToPageList(pageIndex, pageSize, ref totalCount);

// 方式 2：都通用
int i = 1;
var getAll = db.Queryable<Order>().Mapper((it, cache) => {
    it.num = i;  // 有分页需要计算 (pageindex-1)*pagesize+i
    i++;
}).ToList();

// 方式 3: 开窗口函数 (数据库需要支持)
var index = SqlFunc.RowNumber($"{it.Id} asc, {it.Name} desc");
```

**Count 类型为 int 而非 long**：1 亿数据 Count 约需 1 分钟，超过 1000 万行的查询避免查询 Count（耗时超过数据查询本身）。
