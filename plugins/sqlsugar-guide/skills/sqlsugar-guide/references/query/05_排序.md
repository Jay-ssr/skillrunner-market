# 排序

### 基本排序

```csharp
// 升序
db.Queryable<Student>().OrderBy(it => it.Id).ToList();

// 降序
db.Queryable<Student>()
    .OrderBy(it => it.Id, OrderByType.Desc)
    .ToList();

// 多级排序
db.Queryable<Student>()
    .OrderBy(st => st.Id)
    .OrderBy((st, sc) => sc.Id, OrderByType.Desc)
    .ToList();
```

### 多列排序

```csharp
// 混合排序
db.Queryable<Student>()
    .OrderBy(it => new { it.Id, name = SqlFunc.Desc(it.Name) })
    .ToList();
// SQL: id ASC, name DESC

// 多次 OrderBy
db.Queryable<Student>()
    .OrderBy(it => it.Id)
    .OrderBy(it => SqlFunc.Desc(it.Name))
    .ToList();
```

### 动态排序

```csharp
// 1. SQL 字符串排序（防止注入）
db.Queryable<Student>()
    .LeftJoin<School>((st, sc) => st.SchoolId == sc.Id)
    .OrderBy("st.id ASC, sc.Id DESC")
    .Select<ViewModelStudent>()
    .ToList();

// 获取真实列名防注入
EntityMaintenance.GetDbColumnName<Order>("Id")

// 2. 属性名排序（100% 防注入）
db.Queryable<Student>().OrderByPropertyName("Id").ToList();

// 3. 使用合并表排序
db.Queryable<Student>()
    .LeftJoin<School>((st, sc) => st.SchoolId == sc.Id)
    .Select(it => new { .... })
    .MergeTable()
    .OrderByPropertyName("Id")
    .ToList();

// 4. 集合方式排序 (推荐)
List<OrderByModel> orderList = OrderByModel.Create(
    new OrderByModel() { FieldName = "id", OrderByType = OrderByType.Desc },
    new OrderByModel() { FieldName = "name" }  // 默认防注入
);
var list = db.Queryable<Student>()
    .AS("order")
    .OrderBy(orderList)
    .ToList();

// 获取字段名
FieldName = db.EntityMaintenance.GetDbColumnName<Order>("Id");  // 100% 防注入
```

### 随机排序

```csharp
// 随机获取 10 条
db.Queryable<Student>()
    .Take(10)
    .OrderBy(st => SqlFunc.GetRandom())
    .ToList();

// 取最新 10 条
db.Queryable<Student>()
    .Take(10)
    .OrderBy(st => SqlFunc.Desc(it.CreateTime))
    .ToList();
// 或
.OrderBy(it => it.CreateTime, OrderByType.Desc)
```

### OrderByIF

```csharp
// 当条件成立 OrderBy 才生效
db.Queryable<Student>()
    .OrderByIF(IsOrderBy, it => it.Id)
    .ToList();
```

### Case When

```csharp
db.Queryable<Student>()
    .OrderBy(st => SqlFunc.IF(st.type == 1)
        .Return(st.A)
        .ElseIF(st.type == 2)
        .Return(st.B)
        .End(st.C))
    .ToList();
```
