# 分组与去重

### GroupBy 用法

```csharp
// 单字段分组
var list = db.Queryable<Student>()
    .GroupBy(it => it.Name)
    .Select(it => new {
        name = it.Name,
        count = SqlFunc.AggregateCount(it.Id)
    })
    .ToList();

// 多字段分组
var list = db.Queryable<Student>()
    .GroupBy(it => new { it.Id, it.Name })
    .Where(it => it.Id > 0)
    .Select(it => new {
        idAvg = SqlFunc.AggregateAvg(it.Id ?? 0),
        count = SqlFunc.AggregateCount(it.Id),
        name = it.Name
    })
    .ToList();
// SQL: SELECT AVG([Id]) AS [idAvg], COUNT(it.Id), [Name] AS [name]
// FROM [Student] GROUP BY [Name], [Id] WHERE Id > 0

// 带函数分组
.GroupBy(it => SqlFunc.SubString(it.Name, 0, 1))
.GroupBy(it => it.Id)
.GroupBy(it => new { it.Id, name = SqlFunc.ToString(it.Name) })
```

### Having 用法

```csharp
// 聚合对象筛选时使用 Having
var list = db.Queryable<Student>()
    .GroupBy(it => new { it.Id, it.Name })
    .Where(it => it.Id > 0)           // 普通过滤
    .Having(it => SqlFunc.AggregateCount(it.Id) > 0)  // 聚合函数过滤
    .Select(it => new {
        idAvg = SqlFunc.AggregateAvg(it.Id ?? 0),
        count = SqlFunc.AggregateCount(it.Id),
        name = it.Name
    })
    .ToList();
```

### Null 处理

```csharp
// 库中存在 null 时 avg 和 sum 将查不到数据
// AggregateSumNoNull = sum(isnull(num, 0))
SqlFunc.AggregateSumNoNull(it.num)

// AggregateAvgNoNull (5.1.4.108-preview31+) - 等于 avg(isnull(num, 0))
SqlFunc.AggregateAvgNoNull(it.num)

// nullable 类型也可以用 ??
SqlFunc.AggregateSum(it.num ?? 0)

// 原始用法
SqlFunc.AggregateSum(SqlFunc.Isnull(it.num, 0))
```

### Distinct 去重

```csharp
// 指定字段去重
var list = db.Queryable<Student>()
    .Distinct()
    .Select(it => new { it.Name })
    .ToList();
// SQL: SELECT DISTINCT [Name] AS [Name] FROM [STudent]

// Count(Distinct 字段)
db.Queryable<Order>()
    .Select(it => SqlFunc.AggregateDistinctCount(it.Id))
    .ToList();

// 字符串方式
db.Queryable<Order>()
    .Select<int>("count(distinct id)")
    .ToList();
```

### 分组获取第一条

```csharp
// 方式 1：所有数据库通用写法
var list = db.Queryable<Order>()
    .GroupBy(it => it.Name)  // MergeTable 之前不要有 OrderBy
    .Select(it => new {
        name = it.Name,
        id = SqlFunc.AggregateMax(it.Id)
    })
    .MergeTable()
    .LeftJoin<Order>((a, b) => a.id == b.Id)
    .Select((a, b) => b)
    .ToList();

// 方式 2：开窗函数 (5.1.1)
var test48 = db.Queryable<Order>()
    .Select(it => new {
        index2 = SqlFunc.RowNumber(it.Id, it.Name),  // partition by name order by id
        price = it.Price,
        date = it.CreateTime
    })
    .MergeTable()
    .Where(it => it.index2 == 1)  // 相同的 name 只取一条记录
    .ToList();
// SQL: SELECT * FROM (
//   SELECT row_number() over(partition by [Name] order by [Id]) AS [index2],
//          [Price] AS [price], [CreateTime] AS [date]
//   FROM [Order]
// ) MergeTable WHERE [index2] = 1

// 方式 3：个别数据库独有实现
db.Queryable<Order>()
    .Take(1)
    .PartitionBy(it => it.Name)
    .ToList();

db.Queryable<Order>()
    .OrderBy(it => it.id, OrderByType.Desc)
    .Take(1)
    .PartitionBy(it => it.Name)
    .ToList();
```

### 日期分组

```csharp
// 年月分组
var students = db.Queryable<Order>()
    .GroupBy(it => it.CreateTime.ToString("yyyy-MM"))
    .Select(it => new {
        Time = it.CreateTime.ToString("yyyy-MM"),
        Count = SqlFunc.AggregateCount(it.name)
    })
    .ToList();

// 根据年月日分组
var getOrderBy = db.Queryable<Order>()
    .Select(it => new {
        Id = it.Id,
        Name = it.Name,
        CreateTime = it.CreateTime.Date  // 只取日期
    })
    .MergeTable()
    .GroupBy(it => it.CreateTime)
    .Select(it => new { id = SqlFunc.AggregateMax(it.Id), crate = it.CreateTime })
    .ToList();

// SQL 语句分组
.GroupBy(it => SqlFunc.MappingColumn(default(string), "CONVERT(varchar(10),t.F_OutTime, 120)"))

// 按季分组
SqlFunc.DateValue(DateTime.Now, DateType.Quarter)
```

### Count 条件统计

```csharp
// 通过 sum 加三元实现 Count 加条件统计
var list = db.Queryable<Student>()
    .GroupBy(it => new { it.Name })
    .Select(it => new {
        count = SqlFunc.AggregateSum(it.Id > 10 ? 1 : 0)  // Sum(1) 等于 Count
    })
    .ToList();

// 不分组使用 Count
count = SqlFunc.RowCount()  // count(1) over()
```
