# 执行查询

### 返回 List

```csharp
// 查所有
List<Student> list = db.Queryable<Student>().ToList();

// 分页
int pageIndex = 1, pageSize = 20, totalCount = 0;
var page = db.Queryable<Student>().ToPageList(pageIndex, pageSize, ref totalCount);

// 返回第一个
Student item = db.Queryable<Student>().First(it => it.Id == 1);

// 异步
var list = await db.Queryable<Student>().ToListAsync();
```

### 返回 DataTable

```csharp
DataTable dataTable = db.Queryable<Student>().ToDataTable();

// 多字段
var dt = db.Queryable<Student>()
    .Select(it => new { id = it.Id, name = it.Name })
    .ToDataTable();
```

### 返回 Json

```csharp
var json = db.Queryable<Student>().ToJson();

// 序列化时间格式
// Db.Serializable...

// 等同于 ToList 转 json
```

### 返回字典

```csharp
// 表达式返回字典
Dictionary<string, object> dc = db.Queryable<Order>()
    .ToDictionary(it => it.Id, it => it.Name);

// 多表查询 Key 和 Value 不在一个表
xxx.Select((x, y) => new { xkey = x.id, yvalue = y.name })
    .MergeTable()
    .ToDictionary(it => it.xkey, it => it.yvalue);

// 动态 SQL 返回字典
Dictionary<string, object> dc = db.Queryable<Order>()
    .Select<KeyValuePair<string, object>>("id as id1, name as name1")
    .ToList()
    .ToDictionary(it => it.Key, it => it.Value);

// 返回字典集合
List<Dictionary<string, object>> dcList = db.Queryable<Order>()
    .ToDictionaryList();
```

### 返回 Sql 信息

```csharp
var sqlinfo = db.Queryable<Student>().ToSql();

// 返回实体类字符串
var classString = db.Queryable<Student>()
    .Select(it => new { id = it.id, name = it.name })
    .ToClassString();
```

### 行转列

```csharp
// 返回 DataTable
var test01 = db.Queryable<Order>()
    .ToPivotTable(
        it => it.Id,         // 行字段
        it => it.Name,        // 列字段
        it => it.Sum(x => x.Price));  // 值

// 返回 List<dynamic>
var test02 = db.Queryable<Order>()
    .ToPivotList(it => it.Id, it => it.Name, it => it.Sum(x => x.Price));

// 返回 Json
var test03 = db.Queryable<Order>()
    .ToPivotJson(it => it.Id, it => it.Name, it => it.Sum(x => x.Price));

// 空值处理
it => it.Count() > 0 ? it.Sum(x.Price) : 0

// 多列
it => new { name, id }
```

### ThenMapper 延迟映射

对已查询出的列表中的每个元素执行额外的数据库查询，常用于填充关联数据。底层自动按 200 条分批处理。

```csharp
// 查出订单列表
var orders = db.Queryable<Order>().ToList();

// 对每个订单追加查询其明细
db.ThenMapper(orders, order =>
{
    order.Items = db.Queryable<OrderItem>()
        .Where(i => i.OrderId == order.Id)
        .ToList();
});

// 异步版本
await db.ThenMapperAsync(orders, async order =>
{
    order.Items = await db.Queryable<OrderItem>()
        .Where(i => i.OrderId == order.Id)
        .ToListAsync();
});
```

与 `Mapper` / `Includes` 导航查询的区别：
- `Includes` — 在单次 SQL 中 JOIN 或子查询加载，性能最优
- `Mapper` — 在 `ToList()` 执行时自动触发，与查询绑定
- `ThenMapper` — 在任意已存在的列表上延迟执行，灵活度最高，适合复杂场景（如缓存数据补填）

> 内部实现：使用 `Utilities.PageEach(list, 200, ...)` 分批执行，避免一次性发送过多查询。
