# 生命周期

### 执行时机

```csharp
// Queryable 变成非 Queryable 类型时触发 SQL 执行
db.Queryable<Student>().ToList();
```

### 引用类型

```csharp
// 连续操作 Where 条件会叠加
var able = db.Queryable<Student>();
able.Where(it => it.id > 0);
able.Where(it => it.tid > 0);
var list = able.ToList();
// SQL: WHERE id>0 AND tid>0
```

### 拷贝机制

```csharp
// Queryable 复用到多个地方时需要 Clone
var query = db.Queryable<Student>().Where(it => it.Id == 1);

if (条件)
    query.Where(it => it.Name == "jack");

if (条件)
    query.Where(it => it.Id == 1);

// 复用 query 时加 Clone
int count = query.Clone().Count();
var list = query.Clone().ToList();
```
