# 树型查询

### 基础树型查询

```csharp
// 实体
public class Tree
{
    [SqlSugar.SugarColumn(IsPrimaryKey = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    public int ParentId { get; set; }  // 父级字段

    [SqlSugar.SugarColumn(IsIgnore = true)]
    public List<Tree> Child { get; set; }
}

// 第三个参数为 0 表示 parentid 为 0 设置成根目录
var tree = db.Queryable<Tree>().ToTree(it => it.Child, it => it.ParentId, 0);

// 异步
var tree = await db.Queryable<Tree>().ToTreeAsync(it => it.Child, it => it.ParentId, 0);

// 树型权限过滤 (5.1.3.28)
object[] inIds = new object[] { 11, 12 };
db.Queryable<Tree2>().ToTree(it => it.Child, it => it.ParentId, 0, inIds);
```

### 查询所有上级

```csharp
// 获取主键为 11 的所有上级对象
db.Queryable<Tree>().ToParentList(it => it.ParentId, 11);

// 非主键关联 (5.0.7.9-preview05)
// 设置 SugarColumn(IsTreeKey = true)
[SugarColumn(IsTreeKey = true)]
public string Code { get; set; }
```

### 查询所有下级

```csharp
// ToChildList 是没有层级的
var allchilds2 = db.Queryable<Tree>()
    .ToChildList(it => it.ParentId, 2);  // 2 是主键
```

### 非主键映射

```csharp
// 方式 1：指定 id 和 parentid
db.Queryable<Tree>().ToTree(it => it.Child, it => it.ParentCode, 0, it => it.Code)

// 方式 2：通过特性
public class Tree2
{
    [SugarColumn(IsPrimaryKey = true)]
    public int Id { get; set; }
    public string Name { get; set; }

    [SugarColumn(IsTreeKey = true)]  // 设置关联字段
    public string Code { get; set; }
    public string ParentCode { get; set; }

    [SqlSugar.SugarColumn(IsIgnore = true)]
    public List<Tree2> Child { get; set; }
}

db.Queryable<Tree2>().ToTree(it => it.Child, it => it.ParentCode, 0)
```

### 树+联表

```csharp
db.Queryable<Tree>()
    .LeftJoin<Table>((it, t) => it.TreeId == t.Id)
    .Select((it, t) => new ViewModel() {
        ParentId = it.ParentId,
        Id = it.Id,
        TName = t.Name
    })
    .ToTree(it => it.Child, it => it.ParentId, 0, it => it.Id)
```

### 树+导航

```csharp
public class Tree1
{
    [SqlSugar.SugarColumn(IsPrimaryKey = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    public int ParentId { get; set; }

    [Navigate(NavigateType.OneToOne, nameof(ParentId))]
    public Tree1 Parent { get; set; }

    [SugarColumn(IsIgnore = true)]
    public List<Tree1> Child { get; set; }
}

var list = db.Queryable<Tree1>()
    .Includes(x => x.Parent)
    .ToTree(x => x.Child, x => x.ParentId, 0);
```
