# 查询过滤器

> 与 `runtime/26_查询过滤器.md` 内容基本相同，优先读该文件即可。

### 表过滤器

```csharp
// 配置
SqlSugarClient Db = new SqlSugarClient(new ConnectionConfig() {
    ConnectionString = "连接字符串",
    DbType = DbType.SqlServer,
    IsAutoCloseConnection = true
}, db => {
    // 过滤器写在这儿
    db.QueryFilter.AddTableFilter<IDeleted>(it => it.IsDeleted == false);

    // 多租户
    // db.GetConnection(XXX).AddTableFilter
});

// 使用
db.QueryFilter.AddTableFilter<Order>(it => it.Name.Contains("a"));

db.QueryFilter.AddTableFilterIF<Order>(isAdmin == false,
    it => it.Name.Contains("a"));

// 接口过滤器（继承接口的类都有效）5.1.3.47
db.QueryFilter.AddTableFilter<IDeleted>(it => it.IsDelete == false);

// 效果
db.Queryable<Order>().ToList();
// SQL: SELECT ... FROM [Order] WHERE ([Name] LIKE '%'+@MethodConst0+'%')

db.Queryable<OrderItem>()
    .LeftJoin<Order>((i, o) => i.OrderId == o.Id)
    .Select((i, o) => i).ToList();
// SQL: SELECT i.* FROM [OrderDetail] i
// Left Join [Order] o ON (...) AND ([o].[Name] LIKE '%'+@MethodConst1+'%')
```

### 清空过滤器

```csharp
// 单条语句清空
db.Queryable<Order>().ClearFilter().ToList();  // 所有过滤器都无效
db.Queryable<Order>().ClearFilter<IDeleted>().ToList();  // 只有 IDeletedFilter 无效
db.Queryable<Order>()
    .ClearFilter<IDeleted, ITenantFilter>()
    .ToList();  // 多个过滤器无效

// 当前上下文清空
db.QueryFilter.Clear();
db.QueryFilter.Clear<IDeleted>();

// 清空并还原
db.QueryFilter.ClearAndBackup();
db.Queryable<Order>().ToList();
db.QueryFilter.Restore();  // 还原
```

### 联表过滤器

```csharp
// 追加在 ON 后面
db.QueryFilter.AddTableFilter<Order>(it => it.Id == 1);

// 追加在 WHERE 后面
db.QueryFilter.AddTableFilter<Order>(it => it.Id == 1,
    QueryFilterProvider.FilterJoinPosition.Where);
```

### 动态过滤器

```csharp
// 5.1.4.106-preview13+
// 程序启动时注册
StaticConfig.DynamicExpressionParserType = typeof(DynamicExpressionParser);

// 调用
db.QueryFilter
    .AddTableFilter(typeof(IUser), "it", $"it=>it.UserName={"jack"}")
    .AddTableFilter(typeof(IOrg), "it", $"it=>it.OrgId={1} || it.OrgId={2}");
```

### 修改删除使用过滤器

```csharp
// 局部设置
// 删除
db.Deleteable<Order>()
    .EnableQueryFilter()
    .Where(it => it.Id == 0)
    .ExecuteCommand();

// 表达式更新
db.Updateable<Order>()
    .EnableQueryFilter()
    .SetColumns(it => it.Name == "a")
    .Where(it => it.Id == 1)
    .ExecuteCommand();

// 全局设置 (5.1.4.62)
new ConnectionConfig() {
    DbType = DbType.SqlServer,
    ConnectionString = Config.ConnectionString,
    IsAutoCloseConnection = true,
    MoreSettings = new ConnMoreSettings {
        IsAutoDeleteQueryFilter = true,  // 启用删除查询过滤器
        IsAutoUpdateQueryFilter = true    // 启用更新查询过滤器
    }
}
```

### 不依赖实体过滤器

```csharp
// 表单 Queryable 查询
db2.QueryFilter.Add(new SqlFilterItem() {
    FilterValue = it => {
        return new SqlFilterResult() { Sql = " name LIKE '%a%'" };
    },
    IsJoinQuery = false  // 单表生效
});

// 多表 Queryable 查询
db2.QueryFilter.Add(new SqlFilterItem() {
    FilterValue = it => {
        return new SqlFilterResult() { Sql = " o.name LIKE '%a%'" };
    },
    IsJoinQuery = true  // 多表生效
});
```
