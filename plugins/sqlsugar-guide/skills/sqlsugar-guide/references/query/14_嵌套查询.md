# 嵌套查询

### Queryable 和表 JOIN

```csharp
// 写法 1
var queryable = db.Queryable<Order>();
var list = db.Queryable(queryable)
    .LeftJoin<OrderDetails>((o, d) => o.id == d.orderid)
    .Select(o => o).ToList();

// 写法 2
db.Queryable<Order>()
    .Select<Order>()  // MergeTable 之前要有 Select
    .MergeTable()    // 合并成一个表
    .LeftJoin<OrderDetails>((o, d) => o.id == d.orderid)
    .Select(o => o).ToList();

// 写法 3 (5.1.3.41)
db.Queryable<Order>()
    .Where(it => o.id == 1)
    .LeftJoin<OrderDetails>((o, d) => o.id == d.orderid)
    .Select(o => o).ToList();  // 只有 Where 可以不写 MergeTable
```

### 表和 Queryable JOIN

```csharp
var queryableRight = db.Queryable<OrderItem>().Where(x => x.OrderId > 0);
var list = db.Queryable<Order>()
    .LeftJoin(queryableRight, (o, d) => o.Id == d.OrderId)
    .Select(o => o).ToList();
```

### 多个 Queryable JOIN

```csharp
// 新语法
var q11 = db.Queryable<Order>().Where(it => it.Id > 1);
var q22 = db.Queryable<Order>().Where(it => it.Id > 2);
var q33 = db.Queryable<Order>().Where(it => it.Id > 3);

var list = q11.LeftJoin(q22, (x, y) => x.Id == y.Id)
    .LeftJoin(q33, (x, y, z) => x.Id == z.Id)
    .ToList();

// 老语法（最多支持 4 个 Queryable JOIN）
var query1 = db.Queryable<Student, School>((st, sc) => 
    new JoinQueryInfos(JoinType.Left, st.SchoolId == sc.Id))
    .Where(st => st.Name == "jack");

var query2 = db.Queryable<DataTestInfo>();

db.Queryable(query1, query2, JoinType.Left, (p1, p2) => p1.Id == p2.Int1)
    .Select<ViewModelStudent>()
    .ToList();
```

### 表和 SQL JOIN

```csharp
// SqlQueryable 将 SQL 转成 Queryable
var list = db.SqlQueryable<Student>("select * from student")
    .Where(it => it.Id == 1)
    .ToPageList(1, 2);

// 具体用例
db.Queryable<Order>()
    .LeftJoin(db.SqlQueryable<Custom>("select * from Custom"), (o, i) => o.CustomId == i.Id)
    .Select(o => o).ToList();
```

### 表和 List JOIN

```csharp
var queryable = db.Reportable(List<实体>).ToQueryable();
var list = db.Queryable<Order>()
    .InnerJoin(queryable, (x1, x2) => x1.Id == x2.OrderId)
    .Select((x1, x2) => new { name = x1.Name, id = x1.Id, orderid = x2.OrderId })
    .ToList();
```

### 匿名类型嵌套联表

```csharp
// 单个匿名对象 JOIN
db.Queryable<Order>()
    .Select(it => new { id = it.Id })
    .MergeTable()
    .LeftJoin<OrderItem>((x, y) => x.id == y.ItemId)  // 最后一个表不能是匿名对象
    .ToList();

// 多个匿名对象 JOIN
db.Queryable<Order>()
    .Select(it => new { id = it.Id })
    .MergeTable()
    .LeftJoin<OrderItem>((x, y) => x.id == y.ItemId)
    .Select((x, y) => new { xid = x.id, yid = y.ItemId })
    .ToList();
```

### 多合一查询

```csharp
// 多表查询后变成单表查询，不需要加别名
var pageJoin = db.Queryable<Student, School>((st, sc) => 
    new JoinQueryInfos(JoinType.Left, st.SchoolId == sc.Id))
    .Select((st, sc) => new {
        id = st.Id,
        name = sc.Name
    })
    .MergeTable()
    .Where(it => it.id == 1)
    .OrderBy("name ASC")
    .ToList();

// SQL
SELECT * FROM
(SELECT [st].[Id] AS [id], [sc].[Name] AS [name]
FROM [Student] st Left JOIN [School] sc ON ([st].[SchoolId] = [sc].[Id])) MergeTable
WHERE ([id] = @id0) ORDER BY name ASC
```

### 一合一嵌套

```csharp
// 三种写法结果一样
// 写法 1
var subQuery = db.Queryable<Order>().Select(it => new xxx { .... });
var list = db.Queryable(subQuery).ToList();

// 写法 2 (高版本)
var list = db.Queryable<Order>()
    .SelectMergeTable(it => new xxx { .... })
    .ToList();

// 写法 3
var list = db.Queryable<Order>()
    .Select(it => new xxx { .... })
    .MergeTable()
    .ToList();
```
