# 基础查询

## Queryable

`Queryable` 是 SqlSugar 的核心查询类型，类似 EF 的 `IQueryable`，查询从 `db.Queryable<T>()` 开始。

```csharp
var query = db.Queryable<Student>();
```

## 基础用法

```csharp
// 查询所有
List<Student> list = db.Queryable<Student>().ToList();
// SQL: SELECT * FROM Student

// 查询总数
int count = db.Queryable<Student>().Count();
// SQL: SELECT COUNT(1) FROM Student

// 同步 / 异步
int count = db.Queryable<Student>().Where(it => it.Id > 11).Count();
int count = await db.Queryable<Student>().Where(it => it.Id > 11).CountAsync();
```

## 条件查询

```csharp
// 单条件
db.Queryable<Student>().Where(it => it.Id == 1).ToList();
// SQL: SELECT * FROM Student WHERE id = 1

// 多条件 AND
db.Queryable<Student>().Where(it => it.Id > 10 && it.Name == "a").ToList();
// SQL: SELECT * FROM Student WHERE id > 10 AND name = 'a'

// 链式追加条件
db.Queryable<Student>().Where(it => it.Id > 10).Where(it => it.Name == "a").ToList();
// SQL: SELECT * FROM Student WHERE id > 10 AND name = 'a'

// NULL 判断
db.Queryable<Student>().Where(it => it.name != null).ToList(); // IS NOT NULL
db.Queryable<Student>().Where(it => it.name == null).ToList(); // IS NULL
db.Queryable<Student>().Where(it => it.name != "").ToList();  // <> ''
```

## 主键查询

```csharp
// 单主键：SingleById / InSingle
db.Queryable<Student>().InSingle(2);
db.Queryable<Student>().Single(it => it.Id == 2);

// 多主键
var getAll = db.Queryable<Order>()
    .WhereClassByPrimaryKey(new Order() { Pk1 = 1, Pk2 = xx })
    .ToList();

var getAll = db.Queryable<Order>()
    .WhereClassByPrimaryKey(List<Order>)
    .ToList();
```

## First 与 Single

```csharp
// First：返回第一条，没有返回 null（等同于 FirstOrDefault）
db.Queryable<Student>().First(it => it.Id == 1);
// SQL: SELECT TOP 1 * FROM Student WHERE id = 1

// Single：返回唯一一条，多条或无条会抛出异常（等同于 SingleOrDefault）
db.Queryable<Student>().Single(it => it.Id == 1);
// SQL: SELECT * FROM Student WHERE id = 1

// 查询最后一条：先倒序再取第一条
db.Queryable<Student>()
    .OrderBy(it => it.Id, OrderByType.Desc)
    .First(it => it.Id > 10);

// 取前几条
db.Queryable<Student>().Take(10).ToList();
// SQL: SELECT TOP 10 * FROM Student
```

## 聚合统计

```csharp
// 最大值
db.Queryable<Order>().Max(it => it.Id);
await db.Queryable<Order>().MaxAsync(it => it.Id);

// 最小值
db.Queryable<Order>().Min(it => it.Id);
await db.Queryable<Order>().MinAsync(it => it.Id);

// 求和
db.Queryable<Order>().Sum(it => it.Id);
await db.Queryable<Order>().SumAsync(it => it.Id);

// 平均值
db.Queryable<Order>().Avg(it => it.Id);
await db.Queryable<Order>().AvgAsync(it => it.Id);

// 也可用 SqlFunc
SqlFunc.AggregateMax
SqlFunc.AggregateMin
SqlFunc.AggregateSum
SqlFunc.AggregateAvg
```

## EXISTS 判断

```csharp
// 是否存在记录
db.Queryable<Student>().Where(it => it.Id > 11).Any();
// SQL: SELECT CASE WHEN EXISTS(...) THEN 1 ELSE 0 END

// 简化写法
db.Queryable<Student>().Any(it => it.Id > 11); // 异步：AnyAsync()
```

## IN 查询

```csharp
// 单字段 IN
int[] allIds = new int[] { 2, 3, 31 };
db.Queryable<OrderItem>().Where(it => allIds.Contains(it.OrderId)).ToList();
// SQL: orderid IN (2, 3, 31)
// 支持超过 1000，只要数据库不慢没有上限

// 字符串类型：varchar / nvarchar
NameList.Contains(it.Name, true); // true: nvarchar, false: varchar

// 多字段 IN (5.1.4.67-preview04)
List<OrderItem> list = xxx;
db.Queryable<OrderItem>()
    .Where(it => list.Any(s => s.Id == it.Id && s.Name == it.Name));

// IN 模糊查询 (5.1.4.67)
List<OrderItem> list = xxx;
db.Queryable<OrderItem>()
    .Where(it => list.Any(s => it.Name.Contains(s.Name)));

// 数组集合 IN LIKE (5.1.4.85)
List<string> list = xxx;
db.Queryable<OrderItem>()
    .Where(it => list.Any(s => it.Name.Contains(s)));

// NOT IN
db.Queryable<OrderItem>().Where(it => !allIds.Contains(it.OrderId)).ToList();
// SQL: orderid NOT IN (2, 3, 31)
```

## 表名设置

```csharp
// 替换表名
db.Queryable<School>().AS("Student").ToList();
// SQL: SELECT [ID],[NAME] FROM Student

// 添加前缀
db.Queryable<School>().AS("dbo.School").ToList();
// SQL: SELECT [ID],[NAME] FROM dbo.School
```

## 列过滤

```csharp
// 单表：过滤字段
db.Queryable<Order>().IgnoreColumns(it => it.Files).ToList();
// SQL 不会有 Files 列

// 联表：过滤主表
var leftQuery = db.Queryable<Order>().IgnoreColumns(it => it.Files);
var list = db.Queryable(leftQuery)
    .LeftJoin<OrderDetails>((o, d) => o.id == d.orderid)
    .Select(o => o)
    .ToList();

// 联表：过滤 Join 表
var rightQuery = db.Queryable<OrderItem>().IgnoreColumns(it => it.Files);
var list = db.Queryable<Order>()
    .LeftJoin(rightQuery, (o, d) => o.Id == d.OrderId)
    .Select(o => o)
    .ToList();
```
