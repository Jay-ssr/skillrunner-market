# 安装教程

## 按运行时选择包

### .NET Core 3.1 / .NET 5-8

| 类型 | 包名 | 说明 |
|------|------|------|
| 通用版 | `SqlSugarCore` | 内置常用驱动；缺少少见驱动时按提示补装 |
| 纯净版 | `SqlSugarCoreNoDrive` | 不带驱动；按需补装数据库驱动 |

两者代码一致，区别只在 NuGet 是否内置常用驱动。按目标 .NET 版本安装一个包即可。

### .NET Framework

| 类型 | 包名 | 说明 |
|------|------|------|
| 纯净版 | `SqlSugar` | 缺少驱动时按提示补装 |
| 签名版 | `SqlSugar.StrongName` | 用于需要签名的项目 |

## .NET Framework 最低版本

- `5.0.3.3-max` → .NET Framework 4.6
- `5.0.0.2-5.0.3.2` → .NET Framework 4.5
- `4.0-4.9.11` → .NET Framework 4.0+

## 特殊版本

- .NET Core 2.0：安装 `SqlSugarCore_NetCore2`
