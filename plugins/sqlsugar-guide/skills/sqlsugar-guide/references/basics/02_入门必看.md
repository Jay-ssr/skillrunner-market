# 入门必看

## 对象说明

SqlSugar 通过 `SqlSugarClient` 或 `SqlSugarScope` 操作数据库，API 相同，差别在生命周期和线程安全模型。

| 对象 | 模式 | 说明 |
|------|------|------|
| SqlSugarClient | 原生模式 | 高性能；用法类似 Dapper / ADO / EF。不能单例，每个上下文都要新建对象。误用会出现偶发错误；可通过 IOC 或 `db.CopyNew()` 创建新对象 |
| SqlSugarScope | 单例模式 | 对新手更友好；通过 `AsyncLocal` 解决除异步特殊场景外的大多数线程安全问题。遇到 `Task.WhenAll`、`Parallel`、Job 或异步未 `await` 时，仍需 `db.CopyNew()` |

**核心原理**
- `new` 一个 SqlSugar 对象只是创建 C# 对象，本身开销很低
- 实际数据库连接由底层 ADO 连接池管理
- `SqlSugarScope` 只能单例，不能持续 `new`；反复新建会带来内存问题

## 连接参数

`SqlSugarClient` 通过 `ConnectionConfig` 传参。

| 参数名 | 描述 | 必填 |
|--------|------|------|
| DbType | 数据库类型 | 是 |
| ConnectionString | 连接字符串 | 是 |
| IsAutoCloseConnection | 自动释放并关闭数据库连接；事务中在事务结束后关闭，否则每次操作后关闭 | 否 |
| ConfigureExternalServices | 扩展服务集成 | 否 |
| MoreSettings | 更多设置 | 否 |
| SlaveConnectionConfigs | 主从设置 | 否 |
| LanguageType | 错误提示语言 | 否 |

### 超时设置

```csharp
// SQL 超时
db.Ado.CommandTimeOut = 30; // 单位秒

// db.Open 连接超时
// 在连接字符串中追加 Connection Timeout=10，默认 30 秒
```

### 验证连接

```csharp
// 返回 true 表示连接成功
db.Ado.IsValidConnection()

// 如果在事务中验证，建议用新对象，避免影响事务
db.CopyNew().Ado.IsValidConnection()
```

## 打印 SQL

### 单库打印

```csharp
SqlSugarClient Db = new SqlSugarClient(new ConnectionConfig()
{
    ConnectionString = "连接符字串",
    DbType = DbType.SqlServer,
    IsAutoCloseConnection = true
},
db => {
    // 5.1.3.24 起语法与 SqlSugarScope 统一；老版本 AOP 可写在外面
    db.Aop.OnLogExecuting = (sql, pars) =>
    {
        Console.WriteLine(sql); // 输出 SQL，本身对性能无影响

        // 获取原生 SQL，推荐 5.1.4.63
        Console.WriteLine(UtilMethods.GetNativeSql(sql, pars));

        // 获取非参数化 SQL，对性能有影响，尤其是大 SQL / 多参数场景，仅建议调试使用
        Console.WriteLine(UtilMethods.GetSqlString(DbType.SqlServer, sql, pars));
    };
    // 多租户场景需分别配置
    // db.GetConnection(i).Aop
});
```

### 多库打印

```csharp
// 如果使用 GetConnectionScope / GetConnectionScopeWithAttr，AOP 也应使用 GetConnectionScope
// 如果使用 GetConnection / GetConnectionWithAttr，AOP 也应使用 GetConnectionScope

SqlSugarClient Db = new SqlSugarClient(new ConnectionConfig()
{
    ConnectionString = "连接符字串",
    DbType = DbType.SqlServer,
    IsAutoCloseConnection = true
},
db => {
    db.GetConnection("1").Aop.OnLogExecuting = (sql, pars) =>
    {
        Console.WriteLine("执行1库" + sql);
    };
    db.GetConnection("0").Aop.OnLogExecuting = (sql, pars) =>
    {
        Console.WriteLine("执行0库" + sql);
    };
});
```

## 国际化

可通过 `LanguageType` 设置错误提示语言。

```csharp
public enum LanguageType
{
    Default = 0,   // 中&英
    Chinese = 1,   // 已处理异常尽量中文，未处理异常仍可能是英文
    English = 2    // 全部英文
}

SqlSugarClient db = new SqlSugarClient(new ConnectionConfig()
{
    ConnectionString = Config.ConnectionString,
    DbType = DbType.SqlServer,
    LanguageType = LanguageType.English,
    IsAutoCloseConnection = true
});
```

## 长连接用法

| 方式 | 说明 |
|------|------|
| 事务 | 天然长连接 |
| 手动释放模式 | 长连接，需自行释放，不推荐 |
| 自动释放 + `OpenAlways()` | 在当前会话内保持长连接 |

```csharp
// 5.0.6.3
using (db.Ado.OpenAlways())
{
    db.Queryable...
    db.Insertable...
    // 例如依赖当前会话的临时表，需要长连接
}
```

## 配置最小时间

5.0.8.1 新增。推荐保持默认值以兼容多种数据库；也可强制设置。

```csharp
db.CurrentConnectionConfig.MoreSettings = new ConnMoreSettings
{
    DbMinDate = DateTime.MinValue // 默认最小时间是 1900-01-01 00:00:00.000
};
```
