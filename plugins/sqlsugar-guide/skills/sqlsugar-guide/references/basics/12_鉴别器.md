# 鉴别器

## 简介

请升级到 `5.1.4.96-preview10+`，该版本修复了联表查询中的相关问题。

鉴别器（`Discrimator`）类似 TdEngine 的超级表概念：在一张表中存储一个或多个个性化字段；查询时自动附带条件，插入和更新时会自动写入对应值。

**语法格式**
- `Discrimator ="Type:1"`：字段名可自定义
- `Discrimator ="Type:1,Name:a"`：可配置多个字段，注意不要带空格

## 用例

```csharp
[SugarTable("Animal", IsDisabledDelete = true)] // 创建表时若禁止删列需要加上
public class Animal
{
    [SugarColumn(IsIdentity = true, IsPrimaryKey = true)]
    public int AnimalId { get; set; }
    public string Name { get; set; }
}

[SugarTable("Animal", Discrimator = "Type:1", IsDisabledDelete = true)]
public class Dog : Animal
{
    [SugarColumn(IsNullable = true)]
    public int DogId { get; set; }
    [SugarColumn(IsNullable = true)]
    public string Breed { get; set; }
}

[SugarTable("Animal", Discrimator = "Type:2", IsDisabledDelete = true)]
public class Cat : Animal
{
    [SugarColumn(IsNullable = true)]
    public int CatId { get; set; }
    [SugarColumn(IsNullable = true)]
    public string Color { get; set; }
}
```

## 创建表

```csharp
db.CodeFirst.InitTables<Animal, Dog, Cat>(); // 该表会包含所有字段，以及 Type 分类字段
```

## 查询和插入

```csharp
var dog = new Dog { Name = "Buddy", Breed = "Golden Retriever" };
db.Insertable(dog).ExecuteCommand();

var cat = new Cat { Name = "Whiskers", Color = "Gray" };
db.Insertable(cat).ExecuteCommand(); // 实体中没有 Type 字段时，会自动写入特性对应的 Type=2

var catList = db.Queryable<Cat>().ToList(); // 自动加上条件 Type=1
var dogList = db.Queryable<Dog>().ToList(); // 自动加上条件 Type=2
```

## 更新操作

直接更新即可，`Type` 会被忽略，不参与更新。

## 导航场景

导航中也可使用鉴别器：实体里不需要显式声明该字段，但需定义多个类。

```csharp
var dis = db.Queryable<UnitTestDis<Cat>>()
    .Includes(x => x.Animals).ToList(); // T 是 Cat，则导航 Cat

var dis2 = db.Queryable<UnitTestDis<Dog>>()
    .Includes(x => x.Animals).ToList(); // T 是 Dog，则导航 Dog

[SugarTable("UnitTestDis")] // 泛型需要显式指定表名
public class UnitTestDis<T>
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public int Aid { get; set; }
    [Navigate(NavigateType.OneToMany, nameof(Animal.AnimalId), nameof(Aid))]
    public List<T> Animals { get; set; }
}
```
