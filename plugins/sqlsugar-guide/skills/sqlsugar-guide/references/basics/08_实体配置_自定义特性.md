# 实体配置：自定义特性

下面示例演示将自定义特性统一映射到 SqlSugar 配置。

```csharp
SqlSugarClient db = new SqlSugarClient(new ConnectionConfig()
{
    ConnectionString = Config.ConnectionString,
    DbType = DbType.SqlServer,
    IsAutoCloseConnection = true,
    ConfigureExternalServices = new ConfigureExternalServices()
    {
        EntityService = (property, column) =>
        {
            var attributes = property.GetCustomAttributes(true); // 获取全部特性

            if (attributes.Any(it => it is KeyAttribute)) // 根据特性设置主键
            {
                column.IsPrimarykey = true; // 其他属性也可继续在这里统一映射
            }
            // 可写多个规则，便于统一处理并调试
            // if (attributes.Any(it => it is NotMappedAttribute))
            // {
            //     column.IsIgnore = true;
            // }
        },
        EntityNameService = (type, entity) =>
        {
            var attributes = type.GetCustomAttributes(true);
            if (attributes.Any(it => it is TableAttribute))
            {
                var attr = (attributes.First(it => it is TableAttribute) as TableAttribute);
                entity.DbTableName = attr.Name;
            }
        }
    }
});

[Table("student")]
// [SugarTable("student")]
public class MyStudent
{
    [Key]
    // [SugarColumn(IsPrimaryKey = true)]
    public string Id { get; set; }
    public string Name { get; set; }
}
```

该能力不限于示例中的 `KeyAttribute` 和 `TableAttribute`，也可用于统一处理更多自定义特性。
