# 实体配置：无特性配置

## 基础写法

通过规则动态指定主键和自增列，无需在实体上写特性。只要主键和自增信息正确，常规操作即可完成。

```csharp
var db = new SqlSugarClient(new ConnectionConfig()
{
    DbType = SqlSugar.DbType.MySql,
    ConnectionString = Config.ConnectionString,
    IsAutoCloseConnection = true,
    ConfigureExternalServices = new ConfigureExternalServices()
    {
        EntityService = (t, column) =>
        {
            if (column.PropertyName.ToLower() == "id") // 名为 id 的字段设为主键
            {
                column.IsPrimarykey = true;
                if (column.PropertyInfo.PropertyType == typeof(int)) // id 且为 int 时设为自增
                {
                    column.IsIdentity = true;
                }
            }
        },
        EntityNameService = (type, entity) =>
        {
            // entity.DbTableName 修改表名
        }
    }
});
// 可按自身规则设置主键和自增，也可从数据库读取后动态设置
// db.DbMaintenance.GetColumnInfosByTableName 可以获取表信息
```

## 语法糖（5.1.45）

`if / else` 过多时，可用针对指定表的封装，风格接近 Fluent API。

```csharp
SqlSugarClient db = new SqlSugarClient(new ConnectionConfig()
{
    DbType = DbType.SqlServer,
    ConnectionString = Config.ConnectionString3,
    InitKeyType = InitKeyType.Attribute,
    IsAutoCloseConnection = true,
    ConfigureExternalServices = new ConfigureExternalServices()
    {
        EntityService = (s, p) =>
        {
            // Order 实体配置
            p.IfTable<Order>()
            .UpdateProperty(it => it.id, it =>
            {
                it.IsIdentity = true;
                it.IsPrimarykey = true;
            })
            .UpdateProperty(it => it.Name, it => {
                it.Length = 100;
                it.IsNullable = true;
            })
            .OneToOne(it => it.Item, nameof(Order.ItemId));

            // Custom 实体配置
            p.IfTable<Custom>()
            .UpdateProperty(it => it.id, it =>
            {
                it.IsIdentity = true;
                it.IsPrimarykey = true;
            })
            .UpdateProperty(it => it.Text, it => {
                it.DataType = StaticConfig.CodeFirst_BigString; // 兼容多库的大文本写法
            });

            // 可结合全局逻辑一起使用；如有冲突，后面的配置会覆盖前面的
            if (p.IsPrimaryKey == false && new NullabilityInfoContext()
                .Create(c).WriteState is NullabilityState.Nullable)
            {
                p.IsNullable = true;
            }
        },
        EntityNameService = (type, entity) =>
        {
            // entity.DbTableName 修改表名
        }
    }
});

// EntityService 对同一实体只执行一次，通常不必过度担心性能
```
