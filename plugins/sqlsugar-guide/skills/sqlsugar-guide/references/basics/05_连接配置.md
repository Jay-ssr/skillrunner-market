# 连接配置

## 运行时修改示例

可在当前方法中动态修改 `db` 的部分配置，满足局部需求。

### 例 1：临时关闭自动释放

```csharp
db.CurrentConnectionConfig.IsAutoCloseConnection = false;
```

### 例 2：修改 `MoreSettings`

```csharp
// 注意：new ConnMoreSettings() 会清空现有配置
db.CurrentConnectionConfig.MoreSettings = new ConnMoreSettings() { ... };

// 更安全的写法
if (db.CurrentConnectionConfig.MoreSettings != null)
{
    db.CurrentConnectionConfig.MoreSettings.xxxx = xxx;
}
else
{
    db.CurrentConnectionConfig.MoreSettings = new ConnMoreSettings() { ... };
}
```

## 禁止修改的属性

以下属性不应在运行中修改：
- `DbType`：数据库类型
- `ConnectionString`：连接字符串
- `ConfigId`：多租户唯一标识
- `EntityService`、`EntityNameService`：实体 AOP

除此之外，其它属性可按场景调整。

## `MoreSettings` 明细

```csharp
ConnectionString = ...,
DbType = SqlSugar.DbType.MySql,
IsAutoCloseConnection = true,
MoreSettings = new ConnMoreSettings
{
    // 配置项...
}
```

### 二级缓存

| 配置项 | 说明 |
|--------|------|
| IsAutoRemoveDataCache | 控制是否启用自动删除数据缓存；未使用二级缓存时可忽略 |
| DefaultCacheDurationInSeconds | 设置缓存默认持续时间（秒）；未使用二级缓存时可忽略 |

### 查询选项

| 配置项 | 说明 |
|--------|------|
| IsWithNoLockQuery | 启用或禁用 `NOLOCK` 查询提示 |
| IsWithNoLockSubquery | 启用或禁用子查询中的 `NOLOCK` |
| IsAutoUpdateQueryFilter | 控制是否自动更新查询过滤器 |
| IsAutoDeleteQueryFilter | 控制是否自动删除查询过滤器 |

### 字符串处理

| 配置项 | 说明 |
|--------|------|
| DisableNvarchar | 禁用 `NVARCHAR` 数据类型 |
| PgSqlIsAutoToLower | PostgreSQL 查询是否自动转小写；默认小写 |
| PgSqlIsAutoToLowerCodeFirst | PostgreSQL Code First 是否自动转小写；默认小写 |
| IsAutoToUpper | 是否自动转大写；Oracle、达梦、金仓默认大写 |

### 日期时间处理

| 配置项 | 说明 |
|--------|------|
| DisableMillisecond | 禁用毫秒；仅部分数据库支持 |
| DbMinDate | 数据库最小日期，默认 `1900-01-01` |

### 建库建表 / CRUD 相关

| 配置项 | 说明 |
|--------|------|
| EnableOracleIdentity | 启用或禁用 Oracle 自增列；仅 12C+ 支持 |
| SqlServerCodeFirstNvarchar | SQL Server Code First 是否使用 `NVARCHAR` |
| SqliteCodeFirstEnableDefaultValue | SQLite Code First 是否启用默认值 |
| SqliteCodeFirstEnableDescription | SQLite Code First 是否启用备注 |
| SqliteCodeFirstEnableDropColumn | SQLite Code First 是否启用删列 |
| EnableCodeFirstUpdatePrecision | 是否允许 Code First 更新精度 |
| IsCorrectErrorSqlParameterName | 兼容字段特殊字符导致的 SQL 参数错误；会影响性能，可按局部配置启用，如 `db.CurrentConfig.xxx.xx = true` |
| EnableModelFuncMappingColumn | 低代码 API 支持 SQL 函数；非低代码场景可忽略 |
