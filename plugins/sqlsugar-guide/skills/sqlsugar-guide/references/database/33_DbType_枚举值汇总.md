# DbType 枚举值汇总

| MySQL | `DbType.MySql` |
| MySqlConnector | `DbType.MySqlConnector` |
| SqlServer | `DbType.SqlServer` |
| PostgreSQL | `DbType.PostgreSQL` |
| Oracle | `DbType.Oracle` |
| SQLite | `DbType.Sqlite` |
| 达梦 | `DbType.Dm` |
| 人大金仓 | `DbType.Kdbndp` |
| 瀚高 | `DbType.HG` |
| 神通 | `DbType.Oscar` |
| 华为GaussDB | `DbType.GaussDB` / `DbType.GaussDBNative` / `DbType.OpenGauss` |
| Vastbase | `DbType.Vastbase` |
| ClickHouse | `DbType.ClickHouse` |
| TDengine | `DbType.TDengine` |
| MongoDB | `DbType.MongoDb` |
| QuestDB | `DbType.QuestDB` |
| DuckDB | `DbType.DuckDB` |
| Access | `DbType.Access` |
| Doris | `DbType.Doris` |
| GBase | `DbType.GBase` |
| GoldenDB | `DbType.MySql`（需禁用连接池） |
| HANA | 源码使用 |
| OceanBase | `DbType.OceanBase` / `DbType.OceanBaseForOracle` |
| ODBC | `DbType.Odbc` |
| PolarDB | `DbType.PolarDB` |
| TDSQL | `DbType.TDSQL` / `DbType.TDSQLForPGODBC` |
| TiDB | `DbType.Tidb` |
| 自定义 | `DbType.Custom` |
