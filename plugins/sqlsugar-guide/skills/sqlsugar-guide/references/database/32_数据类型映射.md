# 数据类型映射

### 常见数据类型对照

| 数据库类型 | C# 类型 |
|----------|---------|
| blob / binary | `byte[]` |
| clob | `string` |
| number(9,0) | `int` |
| number(18,0) | `long` |
| varchar / nvarchar | `string` |
| date / datetime | `DateTime` |
| timestamp | `DateTime` |
| int / integer | `int` |
| bigint | `long` |
| smallint | `short` |
| decimal / numeric | `decimal` |
| float / double | `double` / `float` |
| boolean / bit | `bool` |
| json | `string` 或可序列化对象（加 `IsJson=true`） |

### JSON 类型

```csharp
[SugarColumn(IsJson = true)]
public List<Order> JsonText { get; set; }
```

### Array 数组类型

```csharp
[SugarColumn(ColumnDataType = "text []", IsArray = true)]
public string[] MenuIds { get; set; }
```

### 自定义类型转换

```csharp
[SugarColumn(SqlParameterDbType = typeof(自定义转换器))]
public TypeName PropertyName { get; set; }
```
