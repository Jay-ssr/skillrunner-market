# 通用配置

## 扩展数据库支持

给 SqlSugar 增加自定义数据库适配，无需修改 `SqlSugar.dll`。

- DEMO：<https://github.com/donet5/SqlSugar>

## 扩展步骤

- 新建类库，例如 `SqlSugar.Access`
- 命名空间改成 `SqlSugar.Access`
- 复制一个最接近的数据库实现
  - 国产库通常更接近 PGSQL
  - Access 通常更接近 SqlServer
- 保留目录结构
  - `Tools`：工具类
  - `Db`：数据库核心实现
- 重命名文件名、类名、命名空间
- 按需要重写继承自基类的方法

## 使用自定义数据库

```csharp
// 程序启动时配置
InstanceFactory.CustomDbName = "Access";
InstanceFactory.CustomDllName = "SqlSugar.Access";
InstanceFactory.CustomNamespace = "SqlSugar.Access";

// 使用自定义数据库
SqlSugarClient db = new SqlSugarClient(new ConnectionConfig()
{
    ConnectionString = Config.ConnectionString,
    DbType = DbType.Custom,
    InitKeyType = InitKeyType.Attribute,
    IsAutoCloseConnection = true
});
```

## 关键点

- `CustomDbName`：文件名前缀
- `CustomDllName`：扩展 DLL 名称
- `CustomNamespace`：扩展 DLL 命名空间
- `DbType.Custom`：启用自定义数据库实现
