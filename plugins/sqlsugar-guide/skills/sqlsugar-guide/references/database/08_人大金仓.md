# 人大金仓

- **NuGet**: `SqlSugarCore` + `SqlSugarCore.Kdbndp`

- **DbType**: `DbType.Kdbndp`

## 连接字符串
```csharp
Server=127.0.0.1;Port=54321;UID=SYSTEM;PWD=system;database=SQLSUGAR4XTEST1
```

## 数据库模式配置
根据版本选择配置：

## R6：Oracle 模式（推荐）
```csharp
SqlSugarClient db = new SqlSugarClient(new ConnectionConfig()
{
    DbType = DbType.Kdbndp,
    ConnectionString = "Server=127.0.0.1;Port=54321;UID=SYSTEM;PWD=system;database=SQLSUGAR4XTEST1",
    IsAutoCloseConnection = true,
    MoreSettings = new ConnMoreSettings()
    {
        DatabaseModel = DbType.Oracle
    }
});
```

## R6：MySQL 模式
```csharp
MoreSettings = new ConnMoreSettings()
{
    DatabaseModel = DbType.MySql
}
```

## R6：PostgreSQL 模式
```csharp
MoreSettings = new ConnMoreSettings()
{
    DatabaseModel = DbType.PostgreSQL
}
```

## R6：SqlServer 模式
```csharp
MoreSettings = new ConnMoreSettings()
{
    DatabaseModel = DbType.SqlServer
}
```
注意：SqlServer 模式下不支持 Date 类型请使用 DateTime 类型。

## R3（比较老版本）
```csharp
SqlSugarClient Db = new SqlSugarClient(new ConnectionConfig()
{
    DbType = DbType.Kdbndp,
    ConnectionString = "Server=127.0.0.1;Port=54321;UID=system;PWD=123;database=test",
    InitKeyType = InitKeyType.Attribute,
    IsAutoCloseConnection = true
}, db => {
    db.Aop.OnExecutingChangeSql = (s, p) =>
    {
        s = s.Replace("pg_catalog.", "");
        return new KeyValuePair<string, SugarParameter[]>(s, p);
    };
});
```

## 注意事项

### 1. 表名是否转大写

默认 `IsAutoToUpper=true` 自动转大写。

不转大写：
```csharp
MoreSettings = new ConnMoreSettings()
{
    IsAutoToUpper = false,  //禁用自动转成大写表
    DatabaseModel = DbType.Oracle
}
```

### 2. 查看安装的模式

```sql
show database_mode;
SELECT version();
```

### 3. 架构支持 Schema

连接字符串上加上 `searchpath=架构名`

### 4. 游标参数

升级：SqlSugarCore 5.1.4.149+
```csharp
//字符串加上
;CursorAsDataRead=True

//具体用法
var p = new SugarParameter("@outcursor", null)
{
    IsRefCursor = true,
    Direction = System.Data.ParameterDirection.Output
};
db.Ado.UseStoredProcedure().ExecuteCommand("cs_20240312", p);
Console.WriteLine(p.Value);
```

### 5. 数组类型

```csharp
//实体
[SugarColumn(IsArray = true)]

//Sql参数
new SugarParameter("Names", array) { IsArray = true }
```

### 6. 特殊类型

#### 6.1 JSON 或者 JSONB：
```csharp
[SugarColumn(IsJson = true)]
```

#### 6.2 Geometry：

NuGet：`SqlSugarCore 5.1.4.189-preview12+` + `Kdbndp.LegacyPostgis_V9`

局部实现：
using (Db.Ado.OpenAlways())
{
    KdbndpConnection conn = (KdbndpConnection)Db.Ado.Connection;
    conn.TypeMapper.UseLegacyPostgis();
    var yyy = Db.Queryable<TestAtt>().ToList();
}

//全局实现
db.Aop.CheckConnectionExecuted = (x, t) =>
{
    KdbndpConnection conn = ((KdbndpConnection)x);
    conn.TypeMapper.UseLegacyPostgis();
};
```

### 7. SM3 加密协议

独立安装最新预览版本的 SqlSugarCore.Kdbndp，SqlSugarCore 5.1.4.160+

### 8. 常见问题

8.1 `sys_config` 表名冲突：避免使用 Sys__Config，以及 `sys_` 和 `pg_` 开头的表名

8.2 `sys_user` 表名冲突：避免使用 Sys_user 表名

8.3 `string =""` 无效：Oracle 模式下没空只有 null，多库用户配置 `ora_input_emptystr_isnull = off`

8.4 `pcolumn.udt_name does not exist`：检查 DatabaseModel 是否设置
