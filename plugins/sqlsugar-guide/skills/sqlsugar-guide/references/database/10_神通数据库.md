# 神通数据库

- **NuGet**: `Oscar.Data.SqlClient` + `SqlSugarCore`

- **DbType**: `DbType.Oscar`

## 连接字符串
```csharp
PORT=2003;DATABASE=osrdb;HOST=localhost;PASSWORD=szoscar55;USER ID=SYSDBA
```

## 初始化
```csharp
SqlSugarClient db = new SqlSugarClient(new ConnectionConfig()
{
    DbType = DbType.Oscar,
    ConnectionString = "PORT=2003;DATABASE=osrdb;HOST=localhost;PASSWORD=szoscar55;USER ID=SYSDBA",
    IsAutoCloseConnection = true
});
```

## 说明
- oscar（KStore 和 MPP 是未来架构，驱动官方还在开发，可以使用 ODBC 方式）
