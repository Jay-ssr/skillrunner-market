# 常见问题汇总

### 连接池问题

部分特殊服务器不支持连接池，在连接字符串加 `Pooling=false`

### Nvarchar 问题

一些数据库不支持 `N'xx'` 语法，配置 `DisableNarvchar=true`

### 大小写问题

- 规范表：自动转大写/小写
- 不规范表：禁用自动转换

### 事务问题

- MySQL 不支持在事务中创建表和删除表
- 部分数据库事务配置不同

### 性能优化

- 大数据写入使用 BulkCopy
- 高并发使用 CopyNew() 保证线程安全
- 合理使用连接池配置

### 字符集问题

- 表情符号使用 utf8mb4
- 特殊字符使用 Nvarchar2

### 驱动问题

- 升级到最新版本解决大多数兼容性问题
- 特殊数据库使用官方驱动
