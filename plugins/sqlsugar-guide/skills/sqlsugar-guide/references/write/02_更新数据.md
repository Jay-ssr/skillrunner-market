# 更新数据

### 实体对象更新

#### 单条与批量

```csharp
//根据主键更新单条
var result = db.Updateable(updateObj).ExecuteCommand(); //实体有多少列更新多少列

//非参数化批量更新
var result = db.Updateable(updateObjs).ExecuteCommand(); //性能好

//参数化批量更新
var result = db.Updateable(updateObjs).UseParameter().ExecuteCommand(); //兼容性好，部分库性能差些

//分页更新 5.1.4.129+
db.Updateable(List<实体>).PageSize(1000).ExecuteCommand();

//只更新修改字段（5.1.4.59支持了批量）
db.Tracking(updateObj); //创建跟踪
updateObj.Name = "a1" + Guid.NewGuid(); //只改修改了name那么只会更新name
db.Updateable(updateObj).ExecuteCommand(); //因为每条记录的列数不一样，批量数据多性能差

//可以清空
db.ClearTracking(); //5.1.4.108-preview30+

//大数据批量更新
db.Fastest<实体>().BulkUpdate(GetList());
```

| 函数 | 说明 |
|------|------|
| ExecuteCommand | 返回受影响行数，update where 如果没找到就会返回 0 |
| ExecuteCommandHasChange | 返回bool，等同于 bool isChange = ExecuteCommand() > 0 |

#### 不更新某列

```csharp
//不更新 TestId和CreateTime
var result = db.Updateable(updateObj)
    .IgnoreColumns(it => new { it.CreateTime, it.TestId })
    .ExecuteCommand();

//也可以用特性
[SugarColumn(IsOnlyIgnoreUpdate = true)]
public DateTime UpdateDate { get; set; }
```

#### 只更新某列

```csharp
//只更新 Name 和 CreateTime
var result = db.Updateable(updateObj)
    .UpdateColumns(it => new { it.Name, it.CreateTime })
    .ExecuteCommand();

//追加AOP赋值列 5.1.4.106-preview19+
var result = db.Updateable(updateObj)
    .UpdateColumns(it => new { it.Name, it.CreateTime }, true) //true表示追加AOP赋值列
    .ExecuteCommand();
```

#### NULL列不更新

```csharp
//更新忽略null字段
db.Updateable(data).IgnoreColumns(ignoreAllNullColumns: true).ExecuteCommand();

//更新忽略null并且忽略默认值(比如int默认值是0就不更新)
db.Updateable(data)
    .IgnoreColumns(ignoreAllNullColumns: true, ignoreAllDefaultValue: true)
    .ExecuteCommand();
```

#### 无主键/指定列更新

```csharp
var result = db.Updateable(updateObj).WhereColumns(it => new { it.Id }).ExecuteCommand(); //更新单条根据ID
var result = db.Updateable(updateObjs).WhereColumns(it => new { it.Id }).ExecuteCommand(); //更新集合根据ID

//可以多列
WhereColumns(it => new { it.Id, it.Name }) //条件列不会被更新，只会作为条件
```

#### 更新添加条件

```csharp
//注意：单条操作都支持，批量只支持部分库
db.Updateable(updateObj).Where(it => it.Id == 1).ExecuteCommand();
```

#### 大数据更新 5.0.4.5

```csharp
//大数据更新，适合大数据更新，可以处理百万级
db.Fastest<实体>().BulkUpdate(GetList()); //特色功能：30列100万8秒更新完
```

#### 重新赋值：list中的值修改

```csharp
//单个字段
db.Updateable(updateObj)
    .ReSetValue(it => { it.Name = it.Name + "a"; })
    .ExecuteCommand();

//多个字段
db.Updateable(updateObj)
    .ReSetValue(it => {
        it.Name = it.Name + "a";
        it.CreateTime = DateTime.Now;
    })
    .ExecuteCommand();
```

#### 重新赋值：列中的值+1

**(1) SET=字段+固定变量 SqlSugarCore 5.1.4.72+**

```csharp
var result = db.Updateable(updateObjs)
    .PublicSetColumns(it => it.Num, it => it.Num + 1) //set num=num+1
    .ExecuteCommand();
```

**(2) SET=字段+集合变量 SqlSugarCore 5.1.4.77 preview02+**

```csharp
db.Updateable(list)
    .PublicSetColumns(it => it.Price, "+") //set price=price+list[i].price
    .ExecuteCommand();
```

### 根据表达式更新（像SQL）

```csharp
var result = db.Updateable<Student>()
    .SetColumns(it => new Student() { Price = it.Price + 1, CreateTime = DateTime.Now })
    .Where(it => it.Id > 11)
    .ExecuteCommand();
```

#### 指定多个字段更新

```csharp
var result = db.Updateable<Student>()
    .SetColumns(it => new Student() { Name = "a", CreateTime = DateTime.Now })
    .Where(it => it.Id == 11)
    .ExecuteCommand();
```

#### 一个字段更新

```csharp
var result = db.Updateable<Student>()
    .SetColumns(it => it.Name == "jack")
    .Where(it => it.Id == 1)
    .ExecuteCommand();
// SQL: Update Student set Name='jack' where id=1
```

#### 字段+1更新

```csharp
var result = db.Updateable<Student>()
    .SetColumns(it => it.Num == it.Num + 1)
    .Where(it => it.Id == 1)
    .ExecuteCommand();
// update Student set iNum=iNum+1 where id=1
```

#### Set语法支持多个

```csharp
var result = db.Updateable<Order>()
    .SetColumns(it => it.Name == it.Name) //生成 [name]=name
    .SetColumnsIF(p != null, it => it.CreateTime == p.Value) //第一条件为true生成 createtime=变量
    .SetColumnsIF(X != null, it => it.X == X) //第一条件为true生成 X=变量
    .Where(it => it.Id == 11)
    .ExecuteCommand();
```

#### 批量更新IN

```csharp
var ids = new int[] { 1, 2, 3 };
var result = db.Updateable<Order>()
    .SetColumns(it => it.Name == "a")
    .Where(it => ids.Contains(it.Id))
    .ExecuteCommand();
```

#### 表达式无实体更新

```csharp
db.Updateable<object>()
    .AS("Order")
    .SetColumns("name", 1)
    .Where("id=1")
    .ExecuteCommand();

//新功能 5.1.4.143+
db.Updateable<object>()
    .AS("UserInfo001")
    .SetColumns(it => SqlFunc.MappingColumn<string>("price"), it => SqlFunc.MappingColumn<string>("price+1"))
    .Where("price=1")
    .ExecuteCommand();
```

#### 表达式分页更新

```csharp
var q = db.Queryable<Order>()
    .Take(10).Skip(10).OrderByDescending(it => it.CreateTime)
    .Select(it => it.Id); //只能是Select单个字段

db.Updateable<Order>()
    .SetColumns(it => new Order() { CreateTime = DateTime.Now })
    .In(it => it.Id, q)
    .ExecuteCommand();
```

### 更多功能

#### 更新主键

ORM 默认不支持修改主键，替代方案：

| 方案 | 操作 |
|------|------|
| 先删后增 | 删除当前记录，插入新记录 |
| 无主键实体 | 新建无主键实体，指定表名用 `WhereColumns` 更新 |

#### 联表更新

```csharp
//多库兼容
var t17 = db.Updateable<Student>()
    .SetColumns(it => new Student() {
        SchoolId = SqlFunc.Subqueryable<School>()
            .Where(s => s.Id == it.SchoolId)
            .Select(s => s.Id),
        Name = "newname"
    })
    .ExecuteCommand();

//优雅写法：MySql PgSql SqlServer Oracle 达梦、金仓
//其中Oracle和达梦只支持2表
var t = db.Updateable<Order>()
    .InnerJoin<Custom>((x, y) => x.CustomId == y.Id)
    .SetColumns((x, y) => new Order() { Name = y.Name, Price = y.Id })
    .Where((x, y) => x.Id == 1)
    .ExecuteCommand();
```

#### 调用实体内方法

```csharp
db.Insertable(new UnitInsertMethod() { ...... })
    .CallEntityMethod(it => it.Create()).ExecuteCommand();

db.Updateable(new UnitInsertMethod() { ..... })
    .CallEntityMethod(it => it.modify("admint")).ExecuteCommand();
```

#### 高性能更新

```csharp
db.Fastest<实体>().BulkUpdate(GetList()); //更新吊打所有框架N倍
```

#### 默认值（系统时间）

**方式1：通过特性指定默认值（5.1.3.42-preview01）**

```csharp
//更新取数据库当前时间 支持多库
[SugarColumn(UpdateServerTime = true, IsOnlyIgnoreInsert = true)]
public DateTime UpdateTime { get; set; }

//更新根据SQL进行插入
[SugarColumn(UpdateSql = "getdate()")]
public DateTime UpdateTime2 { get; set; }

[SugarColumn(UpdateSql = "''")]
public string Str { get; set; }

[SugarColumn(UpdateSql = "0")]
public string Str { get; set; }

[SugarColumn(UpdateSql = "num+1")]
public string num { get; set; }
```

> 注意：
> - 表达更新：SetColumns(it=>new class{ name=it.name},true) 如果用到SetColumn需要加个true
> - 实体更新：无需任何操作
