# 批量操作

### Insertable批量插入


```csharp
// 普通批量插入
var list = new List<Order>();
for (int i = 0; i < 1000; i++)
{
    list.Add(new Order() { Name = "order", Price = i });
}
db.Insertable(list).ExecuteCommand();

// 使用参数化（降低内存）
db.Insertable(list).UseParameter().ExecuteCommand();

// 分批插入
db.Insertable(list).PageSize(100).ExecuteCommand();
```

### Updateable批量更新

```csharp
// 实体批量更新
db.Updateable(list).ExecuteCommand();

// 表达式批量更新
db.Updateable<Order>()
    .SetColumns(it => new Order() { Status = 1 })
    .Where(it => it.CreateTime < DateTime.Now.AddDays(-30))
    .ExecuteCommand();
```

### Deleteable批量删除

```csharp
// 实体批量删除
db.Deleteable(list).ExecuteCommand();

// 条件批量删除
db.Deleteable<Order>()
    .Where(it => it.CreateTime < DateTime.Now.AddDays(-30))
    .ExecuteCommand();
```
