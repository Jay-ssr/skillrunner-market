# 大数据写入

### BulkCopy高速写入

SqlSugar 高效大数据批量写入：

```csharp
// 创建10000条测试数据
var list = new List<Order>();
for (int i = 0; i < 10000; i++)
{
    list.Add(new Order() { Name = "order", Price = i, CreateTime = DateTime.Now.AddHours(i) });
}

// 普通插入
long CommonInsertTime = stopwatch.ElapsedMilliseconds;
db.Insertable(list).ExecuteCommand();

// Fastest极速插入（推荐）
long FastestInsertTime = stopwatch.ElapsedMilliseconds;
db.Fastest<Order>().BulkCopy(list);
```

### 分表BulkCopy

```csharp
// 自动识别分表进行大数据写入
db.Fastest<OrderSpliteTest>().SplitTable().BulkCopy(List<OrderSpliteTest>);

// 固定表大数据写入（不会自动建表）
db.Fastest<OrderSpliteTest>().AS(表名).BulkCopy(List<OrderSpliteTest>);
```

### BulkUpdate分表更新

```csharp
// 分表BulkUpdate
db.Fastest<OrderSpliteTest>().SplitTable().BulkUpdate(List<OrderSpliteTest>);
```

### 部分数据库配置

DM、Oscar 等数据库需要额外配置：

```csharp
// 例如达梦数据库配置
MoreSettings = new ConnMoreSettings()
{
    BulkCopyInsertToDataReader = true  // 启用BulkCopy优化
}
```

### 雪花ID批量处理

雪花ID需手动赋值：

```csharp
foreach (var item in list)
{
    item.Id = SnowFlakeSingle.Instance.NextId();
}
db.Fastest<Order>().BulkCopy(list);
```

### BulkCopy 注意事项

| 事项 | 说明 |
|------|------|
| 目标表 | 使用前确保已存在 |
| 分表 | 使用 `SplitTable()` 自动识别 |
| 雪花ID | 手动赋值后再执行 BulkCopy |
| 性能 | 数据量越大提升越显著 |
