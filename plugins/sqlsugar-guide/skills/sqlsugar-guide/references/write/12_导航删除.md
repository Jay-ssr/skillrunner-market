# 导航删除

### 导航删除语法（多层级）

```csharp
db.DeleteNav<Student>(it => it.Id == 1) //删除主表 Student(id=1)
    .Include(z1 => z1.SchoolA).ThenInclude(z1 => z1.RoomList) //删除2层
    .Include(z1 => z1.Books) //删除1层
    .ExecuteCommand();
```

### 一对一

#### 逻辑

先删子表，再删主表。

```csharp
public class StudentA
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int StudentId { get; set; }
    public string Name { get; set; }
    public int SchoolId { get; set; }

    [Navigate(NavigateType.OneToOne, nameof(SchoolId))]
    public SchoolA SchoolA { get; set; }
}

public class SchoolA
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public string SchoolName { get; set; }
}

//代码：先删SchoolA在删StudentA
db.DeleteNav<StudentA>(it => it.Id == 1)
    .Include(z1 => z1.SchoolA)
    .ExecuteCommand();
```

### 一对多

#### 逻辑

先删子表，再删主表。

```csharp
public class StudentA
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    public int SchoolId { get; set; }

    [Navigate(NavigateType.OneToMany, nameof(BookA.studenId))]
    public List<BookA> Books { get; set; }
}

public class BookA
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int BookId { get; set; }
    public string Name { get; set; }
    public int studenId { get; set; }
}

//代码
db.DeleteNav<StudentA>(it => it.Id == 1)
    .Include(z1 => z1.Books)
    .ExecuteCommand();
```

### 多对多

#### 实体配置

```csharp
public class ABMapping1
{
    [SugarColumn(IsPrimaryKey = true)]
    public int AId { get; set; }
    [SugarColumn(IsPrimaryKey = true)]
    public int BId { get; set; }
}

public class A1
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    
    [Navigate(typeof(ABMapping1), nameof(ABMapping1.AId), nameof(ABMapping1.BId))]
    public List<B1> BList { get; set; }
}

public class B1
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    
    [Navigate(typeof(ABMapping1), nameof(ABMapping1.BId), nameof(ABMapping1.AId))]
    public List<A1> AList { get; set; }
}
```

A 表为主表，B 表为从表，中间表为关系表。

#### 只删中间表（1张表）

```csharp
db.DeleteNav<UserInfo>(x => x.id == 1)
    .Include(x => x.Roles)
    .ExecuteCommand();
```

#### 删A表和中间表（2张表）

```csharp
db.DeleteNav<UserInfo>(x => x.id == 1)
    .Include(x => x.Roles, new DeleteNavOptions() {
        ManyToManyIsDeleteA = true
    })
    .ExecuteCommand();
```

#### 删B表和中间表（2张表）

```csharp
db.DeleteNav<UserInfo>(x => x.id == list4.First().id)
    .Include(x => x.Roles, new DeleteNavOptions() {
        ManyToManyIsDeleteB = true
    })
    .ExecuteCommand();
```

#### 删A、B和中间表（3张表）

```csharp
db.DeleteNav<UserInfo>(x => x.id == 1)
    .Include(x => x.Roles, new DeleteNavOptions() {
        ManyToManyIsDeleteB = true,
        ManyToManyIsDeleteA = true
    })
    .ExecuteCommand();
```

### 自动导航删除（5.1.4.108）

```csharp
db.DeleteNav(list).IncludesAllFirstLayer().ExecuteCommand();
db.DeleteNav(list).IncludesAllFirstLayer(nameof(类.导航), nameof(类.导航2)).ExecuteCommand();

//3级+自动
db.DeleteNav(list)
    .IncludesAllFirstLayer()
    .IncludeByNameString(nameof(类.导航))
    .ThenIncludeByNameString(nameof(类.导航2))
    .ExecuteCommand();
```

### 逻辑删除

逻辑删除只需删主表，子表通过关联查询自动过滤。多对多场景可用导航更新实现中间表逻辑删除。
