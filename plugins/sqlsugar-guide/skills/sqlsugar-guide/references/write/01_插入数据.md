# 插入数据

## 单条插入

```csharp
db.Insertable(insertObj).ExecuteCommand();
await db.Insertable(insertObj).ExecuteCommandAsync();

db.Insertable(insertObj).ExecuteReturnIdentity();
await db.Insertable(insertObj).ExecuteReturnIdentityAsync();

long id = db.Insertable(实体).ExecuteReturnSnowflakeId();
db.Insertable(insertObj).AS("table01").ExecuteCommand();
```

| 方法 | 说明 |
|------|------|
| `ExecuteCommand` | 返回受影响行数 |
| `ExecuteReturnIdentity` | 返回自增列 |
| `ExecuteReturnSnowflakeId` | 返回雪花 ID |

获取数据库时间：`var dbTime = db.GetDate();`

## 批量插入

### 常规批量

```csharp
db.Insertable(List<实体>).ExecuteCommand();
db.Insertable(List<实体>).PageSize(1000).ExecuteCommand();
```

| 场景 | 说明 |
|------|------|
| 通用场景 | 数据量很大时内存占用会变高 |

> 泛型参数是实体类型，不能写成 `Insertable<List<实体>>(List)`。

### 参数化批量

```csharp
db.Insertable(List<实体>).UseParameter().ExecuteCommand();
```

| 场景 | 说明 |
|------|------|
| 条数较少、调用频繁 | 个别数据库 500 条以上可能变慢；需要事务时自行包裹 |

### 大数据写入

```csharp
db.Fastest<实体>().PageSize(100000).BulkCopy(List<实体>);
```

| 场景 | 说明 |
|------|------|
| 1000+ 数据量 | 不支持数据库默认值，API 更精简 |
| MySql | 连接串需要 `AllowLoadLocalInfile=true` |

### `IntoTable`

```csharp
db.Queryable<Order>()
    .Select(it => new { name = it.name, ...... })
    .IntoTable<实体2>();

db.Queryable<Order>()
    .IntoTable<Order>("新表名");
```

目标表是自增列时可忽略该列，但生成的 ID 会不同。

## 其他输入形式

### 字典

```csharp
var dc = new Dictionary<string, object>();
dc.Add("name", "1");
dc.Add("CreateTime", DateTime.Now);
db.Insertable(dc).AS("student").ExecuteCommand();
```

### `DataTable`

```csharp
List<Dictionary<string,object>> dc = db.Utilities.DataTableToDictionaryList(dataTable);
db.Insertable(dc).AS("student").ExecuteReturnIdentity();

db.Fastest<System.Data.DataTable>().AS("order").BulkCopy(dataTable);
```

> `5.0.7.7` 以下 `DataTableToDictionaryList` 存在已知问题。

### 匿名对象 / `object`

```csharp
db.InsertableByDynamic(new { name = "", price = 1 })
    .AS("[Order]")
    .ExecuteCommand();

db.InsertableByObject(o).ExecuteCommand();
```

`InsertableByObject` 要求 `o` 实际上是真实类实例。

## 常用返回值

| 方法 | 说明 |
|------|------|
| `ExecuteCommand` | 返回受影响行数 |
| `ExecuteReturnPkList<T>()` | 批量返回主键，自增/雪花 ID/GUID 均支持 |
| `ExecuteReturnIdentity` | 返回单个自增列 `int` |
| `ExecuteReturnBigIdentity` | 返回单个自增列 `long` |
| `ExecuteReturnEntity` | 返回实体；可回填自增列，不支持批量自增/默认值 |
| `ExecuteCommandIdentityIntoEntity` | 给传入实体回填自增列，不支持批量 |
| `ExecuteReturnSnowflakeId` | 返回雪花 ID |
| `ExecuteReturnSnowflakeIdList` | 返回雪花 ID 集合 |

## 常见控制项

### 忽略列 / 指定列

```csharp
db.Insertable(insertObj)
    .IgnoreColumns(it => new { it.Name, it.TestId })
    .ExecuteReturnIdentity();

db.Insertable(insertObj)
    .InsertColumns(it => new { it.Name, it.SchoolId })
    .ExecuteReturnIdentity();
```

### 忽略空值列

```csharp
db.Insertable(insertObj2)
    .IgnoreColumns(ignoreNullColumn: true)
    .ExecuteCommand();
```

> 不同记录插入列不一致时，不适合批量操作。

### 异步

```csharp
await Db.Insertable(new Order() { Name = "a", CustomId = 1 }).ExecuteCommandAsync();
```

## 导航与大数据

### 导航插入

```csharp
db.InsertNav(list)
    .Include(z1 => z1.SchoolA)
    .ThenInclude(z1 => z1.RoomList)
    .Include(z1 => z1.Books)
    .ExecuteCommand();
```

### 大数据插入

```csharp
db.Fastest<RealmAuctionDatum>().BulkCopy(GetList());
db.Fastest<RealmAuctionDatum>().PageSize(50000).BulkCopy(List<实体>);

db.Insertable(List<实体>).PageSize(1000).ExecuteCommand();
```

## 默认值与特殊行为

### 使用数据库默认值

```csharp
public class Order
{
    [SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }
    [SugarColumn(IsOnlyIgnoreInsert = true)]
    public DateTime CreateTime { get; set; }
}
```

### 用特性指定插入值

```csharp
[SugarColumn(InsertServerTime = true, IsOnlyIgnoreUpdate = true)]
public DateTime CreateTime { get; set; }

[SugarColumn(InsertSql = "getdate()")]
public DateTime CreateTime2 { get; set; }

[SugarColumn(InsertSql = "''")]
public string Str { get; set; }

[SugarColumn(InsertSql = "0")]
public string Str { get; set; }
```

### `Guid` 主键自动赋值

主键类型为 `Guid` 且未传值时，会自动生成。

> 只能配合 `ExecuteCommand`，不能使用自增列返回方法。

## 调用实体方法

```csharp
var data = new UnitInsertMethod() { Name = "a", Time = DateTime.Now };
db.Insertable(data).ExecuteCommand();
db.Updateable(data).CallEntityMethod(it => it.modify("admint")).ExecuteCommand();

public class UnitInsertMethod
{
    [SqlSugar.SugarColumn(IsPrimaryKey = true, IsIdentity = true)]
    public int Id { get; set; }
    public string Name { get; set; }
    public DateTime Time { get; set; }
    [SqlSugar.SugarColumn(IsNullable = true)]
    public string UserId { get; set; }

    public void Create()
    {
        this.Time = DateTime.Now;
        this.UserId = "1";
    }

    public void modify(string a)
    {
        this.Time = DateTime.Now;
        this.UserId = a;
    }
}
```

## 强制插入自增列

```csharp
db.Insertable(insertObj).OffIdentity().ExecuteCommand();
```

需要数据库支持，建议版本 `5.1.4.73-preview12+`。
