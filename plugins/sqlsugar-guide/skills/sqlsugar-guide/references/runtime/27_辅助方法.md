# 辅助方法

## 常用用法

通过 `db.Utilities` 调用：

```csharp
// 例1：DataTable 转 List
var list = db.Utilities.DataTableToList(datatable);

// 例2：序列化对象
db.Utilities.SerializeObject(lists);

// 例3：分页处理内存集合
db.Utilities.PageEach(allList, 100, pageList =>
{
    db.Insertable(pageList).ExecuteCommand();
});
```

## 常见方法

| Name | 说明 |
|------|------|
| DataReaderToExpandoObject | ExpandoObject |
| DataReaderToExpandoObjectList | List<dynamic> |
| DataReaderToList | List<T> |
| SerializeObject | 序列化 |
| DeserializeObject | 反序列化 |
| TranslateCopy | 复制对象 |
| DataTableToDynamic | dynamic |
| DataTableToList | List<T> |
| DataTableToDictionary | Dictionary<string, object> |
| RemoveCacheAll | 清空 ORM 全部缓存，不建议直接使用 |
| PageEach | 内存分页 |
